# ­Ъје CSS Grid Mastery - 10 Levels

рЃЏрЃЮрЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ! рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃЌ **CSS Grid**-рЃўрЃА рЃАрЃарЃБрЃџрЃДрЃЮрЃцрЃўрЃџрЃљрЃЊ рЃљрЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљрЃерЃў. рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ 10 рЃЊрЃЮрЃюрЃўрЃАрЃњрЃљрЃю (Level), рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃЊрЃљрЃю рЃарЃЌрЃБрЃџрЃўрЃАрЃЎрЃћрЃю.

## ­ЪЊѓ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

- `index.html` - рЃљрЃЦ рЃљрЃарЃўрЃА рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃЏрЃ«рЃљрЃарЃћ (HTML). **рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ, рЃљрЃЏ рЃцрЃљрЃўрЃџрЃА рЃ«рЃћрЃџрЃў рЃљрЃа рЃљрЃ«рЃџрЃЮрЃЌ!** РЏћ
- `style.css` - рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃцрЃљрЃўрЃџрЃў. рЃЎрЃЮрЃЊрЃў рЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃЦ. РюЇ№ИЈ

## ­Ъџђ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

1.  рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃЮрЃџрЃЊрЃћрЃарЃў **VS Code**-рЃерЃў.
2.  рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ `style.css` рЃцрЃљрЃўрЃџрЃў.
3.  рЃЕрЃљрЃЏрЃЮрЃДрЃћрЃЋрЃўрЃЌ рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃА. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃџрЃћрЃЋрЃћрЃџрЃА рЃљрЃЦрЃЋрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: `/* CODE HERE */`.
4.  рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ, рЃЕрЃљрЃгрЃћрЃарЃЮрЃЌ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў Grid рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў (properties), рЃарЃљрЃЌрЃљ рЃЏрЃўрЃўрЃдрЃЮрЃЌ рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљ.
5.  рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў `index.html`-рЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃўрЃЌ.

---

## ­ЪЈє рЃЊрЃЮрЃюрЃћрЃћрЃЉрЃў (Levels)

рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ рЃцрЃљрЃарЃљрЃЋрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЌрЃћрЃЏрЃљрЃА:

- [ ] **Level 1:** Grid-рЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃАрЃЋрЃћрЃбрЃћрЃЉрЃў (`grid-template-columns`)
- [ ] **Level 2:** рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў (`grid-template-rows`)
- [ ] **Level 3:** Fraction рЃћрЃарЃЌрЃћрЃБрЃџрЃў (`fr`)
- [ ] **Level 4:** рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃћрЃЉрЃљ (`repeat()`)
- [ ] **Level 5:** рЃЊрЃљрЃерЃЮрЃарЃћрЃЉрЃћрЃЉрЃў (`gap`)
- [ ] **Level 6:** рЃАрЃЋрЃћрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ (`grid-column`)
- [ ] **Level 7:** рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ (`grid-row`)
- [ ] **Level 8:** Grid Areas (`grid-template-areas`)
- [ ] **Level 9:** рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ (`justify-items`, `align-items`)
- [ ] **Level 10:** рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЋрЃАрЃћрЃЉрЃљ (`auto-fit`, `minmax`)

---

­ЪњА **рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ:** рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљрЃерЃў рЃњрЃљрЃўрЃГрЃћрЃЊрЃћрЃЌ, рЃњрЃљрЃўрЃ«рЃАрЃћрЃюрЃћрЃЌ `display: grid;` рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃќрЃћ.

**рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃћрЃЉрЃў!** ­Ъњ╗
